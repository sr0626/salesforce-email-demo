# DNS requests to Server Sea — `ccaas.evolvity.com` (SES)

Server Sea manages the live **Cloudflare** zone for `evolvity.com` and adds records on
our behalf. Every record below is on the **`ccaas.evolvity.com` subdomain** for Amazon
SES (region **us-west-2**), and every one must be **"DNS only" / NOT proxied** (grey
cloud) — a proxied CNAME breaks DKIM.

**Never touch the root `evolvity.com` MX/SPF.** Only **one** SPF (and one DMARC) TXT
record is valid per name — if one already exists on a name, amend it, don't add a second.
